My Games

OBTAINIUM

In the spring of 2023, I took a course called Media Device Architecture. It focused on data manipulation on media devices, and we spent the semester developing games and other applications for the Game Boy Advance. The course culminated in a large scale final project, where we each made a game using the techniques we had learned. Given the low-level focus of the class, this means that every level of implementation, from setting bits in individual pixels and managing the audio buffer to game design and music, was done by myself over the course of about 30 days. All code for the game was written in C. Below is a playthough of the project.

ACQUIFISHING

In the fall of 2023, as a final project for a course called Introduction to Computational Media, I worked with a team on a card game called Acquifishing. I undertook the role of designing the game and implementing the game's code in Javascript. Below is a link to a page where you can play the game yourself!
